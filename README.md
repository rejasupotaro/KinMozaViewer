![](https://raw2.github.com/rejasupotaro/KinMoza/master/screenshot.png)

A refugee camp for alice lovers.

## Installation

Clone this repository:

```
$ git clone git@github.com:takiguchi0817/KinMoza.git
```

And install app:

```
$ ./gradlew installDebug
```
