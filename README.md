KinMozaApp
==========

![](https://dl.dropboxusercontent.com/u/54255753/blog/201309/alice.png)

A refugee camp for alice lovers.

## Installation

Clone this repository:

```
$ git clone git@github.com:takiguchi0817/KinMoza.git
```

And install app:

```
$ ./gradlew installDebug
```
